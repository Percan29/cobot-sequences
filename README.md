# Cobot Sequences – Industrial Automation

This repository contains simulated cobot sequences written in Python, focused on industrial automation, robotics, and manufacturing workflows.  
The goal is to demonstrate core concepts used in real cobot environments such as movement control, welding paths, sensor validation, safety logic, and trajectory planning.

---

##  Repository Structure

### movement/
Linear, joint and circular movement sequences used in cobots (UR, KUKA, FANUC, ABB).

### welding/
Simulated welding paths and repetitive weld cycles used in robotic welding cells.

### sensors/
Basic sensor validation logic for part detection and process confirmation.

### safety/
Emergency stop conditions, temperature limits, and safety-related decision logic.

### trajectories/
Curved and circular trajectories used for polishing, inspection, and arc welding.

---

##  Purpose

This project demonstrates how Python can be used to simulate industrial cobot logic, helping technicians and automation learners understand:

- Movement sequences  
- Welding operations  
- Sensor-based decisions  
- Safety conditions  
- Industrial trajectories  

It is designed as a learning and demonstration tool for industrial automation environments.

# Secuencias para Cobots – Automatización Industrial

Este repositorio contiene secuencias simuladas para cobots escritas en Python, enfocadas en automatización industrial, robótica y procesos de manufactura.  
El objetivo es demostrar conceptos reales utilizados en entornos industriales: control de movimiento, trayectorias de soldadura, validación de sensores, lógica de seguridad y planificación de trayectorias.

---

##  Estructura del Repositorio

### movement/
Secuencias de movimiento lineal, articular y circular utilizadas en cobots (UR, KUKA, FANUC, ABB).

### welding/
Trayectorias de soldadura simuladas y ciclos repetitivos usados en celdas robotizadas.

### sensors/
Lógica básica de validación de sensores para detección de piezas y confirmación de procesos.

### safety/
Condiciones de parada de emergencia, límites de temperatura y decisiones relacionadas con seguridad.

### trajectories/
Trayectorias curvas y circulares usadas para pulido, inspección y soldadura en arco.

---

##  Propósito

Este proyecto demuestra cómo Python puede simular lógica industrial de cobots, ayudando a técnicos y aprendices de automatización a comprender:

- Secuencias de movimiento  
- Operaciones de soldadura  
- Decisiones basadas en sensores  
- Condiciones de seguridad  
- Trayectorias industriales  

Diseñado como herramienta de aprendizaje y demostración para entornos de automatización industrial.
